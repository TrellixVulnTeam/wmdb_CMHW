This is a web interface to project database: Your Movie Database (YMDB). It contains information on movies, actors, directors, and user reviews. Here is the schema for the underlying database:

USER UID, u_name, email, created_date
ADMIN UID, position
DIRECTOR UID, famous_for_MID, given_name, DoB
ACTOR UID, stage_name, given_name, DoB
MOVIE MID, director_UID, title, release_date, entered_by, entered_date
REVIEW MID, UID, text, rating, created_date
ACTED_IN MID, UID, character_role
POSTER MID, img
